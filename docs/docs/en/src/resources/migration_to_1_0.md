# Migrating to VSAG 1.0

> **Status:** draft — content is being filled in as part of the 1.0
> documentation readiness work tracked in
> [issue #2069](https://github.com/antgroup/vsag/issues/2069).

This page collects everything users coming from VSAG 0.18.x (and earlier)
need to know to upgrade smoothly to **VSAG 1.0**. Read it before you
recompile or redeploy.

- Serialization-format compatibility statement.
- Default-value and behavioral changes.
- Build-system / packaging notes.
- Step-by-step upgrade checklist.
